## Decision rules (asymmetric)

**Reposition triggers** (in priority order):
- Price within 15% of boundary → consider repositioning
- Bot has been out of range for >5 days (down) or >1 day (up) → reposition
- ATR has changed >40% from when bot was created → resize range
- Major macro event (CPI, FOMC, NFP) in next 24h → widen range 50%

**Asymmetric repositioning**:
- UP breakout: reposition aggressively (have USDT, no realized loss)
- DOWN breakout: wait first, only force after 5 days or 5%+ below floor
- When repositioning, center NEW range slightly low-biased (current - 0.2 × half_width)

**Range sizing rules**:
- half_width = max(1.5 × ATR(7d), $75)
- if macro event imminent: half_width × 1.5
- Step size: aim for $20-25 (4-5× the 0.1% round-trip fee)
- Number of grids: width / step

**Cost-benefit gate**:
- Each adjustment costs ~$0.30 (cancel + create + small slippage)
- Only adjust if expected profit improvement > $1 over next few days
